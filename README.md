Projeto inspirado pela [Atividade de extensão](AEX.pdf) da professora Leo Sampaio Ferraz Ribeiro, da materia de Algoritmos Avançados.
Escolhemos a opção 2, na qual faremos uma apresentação utilizando teoria dos números no algoritmo de RSA para ser
útil à sociedade.

## Compilação da atividade interativa
Para compilar a atividade interativa, os seguintes passos são necessários:
1- Baixe o projeto no github:
https://github.com/GuilhermePereiraSa/projetoAlgAvan/
2- Baixe a Unity Engine (versão 6000.0.24f1):
https://unity.com/pt/download
3- Abra o projeto
4- Selecione na barra de ferramentas: 
File > Build and Run
